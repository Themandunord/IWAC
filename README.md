# IWAC

> Command line tool to create Watson Assistant Workspaces

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i iwac -g
```

> Keep in mind that Watson Assistant have a Rate Limit of 30 requests every 30 Minutes

## Usage

```js
iwac --help
iwac create (alias c)
iwac delete (alias d)
```
